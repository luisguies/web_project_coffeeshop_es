**Nombre de Proyecto:**
*Coffeeshop Web Project*
-----------------------

**Descripcion del Proyecto:**
Este proyecto fue hecho para una cafeteria llamada Triple Espresso. Se hizo una pagina que incluye lo siguiente:


*Header con:*
1. Logo
2. Navbar
3. Titulo con imagen
4. Horario y dirección.

*Main con:*
1. titulo "Reserva una mesa"
2. Un formulario para llenar con los siguientes atributos:
a. Nombre
b. Numero de comensales
c. Fecha y hora
d. correo electronico
e. boton para hacer la reserva
f. checkbox para aceptar terminos y condiciones.
g. labels para cada una de las opciones.

*Footer con:*
1. Logo en otro color
2. Forma geometrica (circulo)
3. conexion a redes sociales
4. el ©
